# browser-static-features

浏览器端实用工具集，纯前端实现，无需后端服务。

## 🎬 video-convert

浏览器端视频格式转换工具，使用 ffmpeg.wasm 实现，可将任意视频转换为 H264 MP4。

**直接用浏览器打开 `video-convert/index.html` 即可使用。**

### 功能特性

- 🖥️ 纯浏览器端转换，无需上传服务器
- 🎬 输出 H264 MP4，兼容所有浏览器和平台
- 📦 支持格式：mp4, webm, avi, mkv, mov, flv, wmv, m4v, 3gp, ogv
- ⚡ 支持自定义视频/音频码率
- 📊 实时转换进度显示

### 使用方法

1. 用浏览器直接打开 `video-convert/index.html`
2. 拖拽或点击上传本地视频文件
3. 等待转换完成，点击下载
